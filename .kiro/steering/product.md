# Fashion Store - Product Overview

A modern, full-featured fashion e-commerce website with comprehensive authentication system. Built as a complete online fashion store with user registration, email verification, secure login, shopping cart, and checkout functionality.

## Key Features
- Complete e-commerce functionality with product catalog, search, and filtering
- Comprehensive authentication system with email verification
- Mobile-first responsive design with SEO optimization
- Vietnamese language support for email templates and user interface
- Admin tools for testing and monitoring
- Security-focused with rate limiting, input sanitization, and JWT authentication

## Target Audience
Fashion retailers and e-commerce businesses looking for a complete, secure online store solution with modern UX/UI and robust authentication.

## Business Logic
- User registration with email verification using 4-digit codes
- Secure password hashing with bcrypt
- JWT-based session management
- Shopping cart and checkout flow
- Product catalog with advanced search and filtering
- Admin monitoring and testing interfaces
